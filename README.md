# To-Do App

Aplicación web de lista de tareas desarrollada con HTML, CSS y JavaScript puro.

## Funcionalidades
- Agregar tarea
- Marcar tarea como completada
- Eliminar tarea
- Contador de tareas pendientes
- Validación para evitar tareas vacías o con espacios

## Tecnologías
- HTML
- CSS
- JavaScript (Vanilla JS)
- Manipulación del DOM
